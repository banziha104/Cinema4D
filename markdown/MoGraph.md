# MoGraph

> 모그라프때 늦었다. 연습해라 미래의 영준아 

- Motext :

- Effector
    - Shader      
    - Random : 랜덤으로 생성
    - Step : 단계별로 들어감
    - Delay : 지연시킴
    - Spline Effect
    - Inheritance : 계승, 움직임을 뻇어오거나 모양을 변형
    - Push Apart
    - ReEffector : 이펙트를 다시줌
    - Spline 
    - Sound : 사운드에 반응해서 모그라프가 움직임
    - Target : 바라보게함
    


- Voronoi Fracture
- Cloner 
    - Mode -> Object : 오브젝트에 맞춰서 만듬
    - Distribution -> Volume : 해당 하는 객체 안에 랜덤으로 생성
- Tracer : 발자취를 남김
    - Object
        - Tracing Mode  
        - Limit : 선의 제한사항
